Download Navi es un gestor de descargas gratuito y de código abierto para Android 4.4+.

# Características

* Software gratuito y de código abierto
* Material Design
* Múltiples temas (Claro, Oscuro, Negro)
* Soporte para Android TV
* Soporte para Chrome OS
* Pausar y reanudar descargas
* Soporte para navegadores
* Descargas simultáneas (3 por defecto)
* Descarga de partes múltiples (máximo 16)
* Descargar archivos en segundo plano
* Soporte solo para conexiones sin medidor (ejemplo Wi-Fi)
* Controles de itinerancia
* Controles de agente de usuario
* Calcular el valor de hash (MD5 y SHA-256)
* Administración de energía y control de batería
* Diversas acciones automáticas tras finalizar la descarga
